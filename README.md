Este repositorio contiene una aplicación profesional de **Streamlit** diseñada para el análisis exploratorio de la calidad del agua, basada en datos de potabilidad de Kaggle.

### 🚀 Funcionalidades
* **Landing Page:** Introducción al proyecto y métricas clave iniciales.
* **Dashboard Interactivo:** Filtros dinámicos y visualizaciones avanzadas con Plotly.
* **Integración de Datos:** Procesamiento eficiente de archivos CSV.

### 🛠️ Instalación y Uso
Para ejecutar este proyecto localmente:

1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`.
3. Ejecutar la app: `streamlit run app.py`.

### 📊 Dataset
El dataset utilizado se puede obtener vía Kaggle con el comando:
`kaggle kernels pull kattpg/an-lisis-exploratorio-calidad-del-agua`
